# ArduinoSys
A OS for the Arduino <br>
# Setup
If your Arduino has no USB port, you need to solder a PS/2 port <br>
The pins you solder are: <br>
Data > D2 <Br>
Clock > D3 <Br>
VCC > 5V <br>
GND > GND <Br>
Solder a OLED screen to these pins: <br>
VCC > 5V <br>
GND > GND <br>
SDA > A4 <br>
SCL > A5 <br>
Solder 2 buttons to these pins: <Br>
D6 (Left-Click) <Br>
D7 (Right-Click) <Br>
